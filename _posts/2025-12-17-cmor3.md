Changes
* The [CMIP7 CMOR tables repo](https://github.com/WCRP-CMIP/cmip7-cmor-tables){:target="_blank"} will now be a submodule of the CMOR repo. CMIP7 unit tests for CMOR have been modified to use these tables.
* The attribute `branded_variable` will be added to a dataset's global attributes if the variable table entry name has a `branding_suffix`.
